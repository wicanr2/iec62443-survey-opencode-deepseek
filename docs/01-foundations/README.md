# 01 基礎概念

在進入軟硬體的技術要求（4-1、4-2）之前，你需要先理解三件事：
1. 為什麼 OT（工控）需要自己的一套資安標準，不能直接套 IT 的？
2. 在一個工廠裡，信任邊界怎麼劃？（Zone & Conduit）
3. 「夠安全」是多安全？怎麼量化？（Security Levels）
4. 安全的面向有哪些？怎麼分解？（FR 1-7）
