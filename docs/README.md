## 範圍說明：這份知識庫涵蓋標準族的哪一塊

IEC 62443 是一個 **4 群主體 + 評估方法學** 的大家族（完整全景見 [01 全景圖](01-foundations/01-iec62443-overview.md)）。本庫**核心聚焦產品供應商視角**——把 **-4-1（安全開發）** 與 **-4-2（組件技術要求）** 講到能落地，是 01–05 章。

| 章 | 對應標準 | 視角 |
|---|---|---|
| 01–05（核心） | -4-1、-4-2，輔以 -3-3/-3-2 上下文 | **產品供應商**：做出並認證一個安全組件 |
| 06（延伸） | -3-3、-3-2、-2-1、-1-6 | **超出核心**：系統整合商 / 資產擁有者 / IIoT 場景 |

刻意**不深入**的部分（點到即止，需要時去 IEC 原文）：-2-x 業主/服務商完整管理計畫、-3-1 安全技術總覽、-1-x 多為制定中。這是取捨，不是遺漏——你的產品要拿 CSA，主戰場在 -4-1/-4-2。
